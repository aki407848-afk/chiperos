# 🛡️ ChiperOS v1 BETA
**Educational Multi-Tool Firmware for ESP32-S3**

---
## 🇷🇺 О Проекте (RU)
**ChiperOS** — это экспериментальная операционная система для микроконтроллера ESP32-S3, разработанная в образовательных целях. Проект предназначен для изучения протоколов связи, анализа радиочастотного спектра и отладки устройств умного дома.

### ⚡ Возможности:
*   **Интерфейс:** Полностью настраиваемый UI на экране ST7789 (1.14").
*   **WiFi & BLE:** Точка доступа, анализатор трафика, управление соединениями.
*   **RF Sub-GHz:** Сниффинг и анализ сигналов на 433/868 МГц (CC1101).
*   **IR Protocol:** Запись, воспроизведение и подбор ИК-сигналов (DHT11/кондиционеры/ТВ).
*   **Web Control:** Встроенный веб-сервер для мониторинга через браузер.
*   **Safety System:** Watchdog и Safe Mode для защиты от зависаний.

📦 Комплектация:
  ESP32-S3 DevKit
   Дисплей ST7789 1.14" (SPI)
   CC1101 / NRF24L01 модули
   ИК-приемник/передатчик

## 🇺🇸 About Project (EN)
**ChiperOS** is an experimental firmware for the ESP32-S3 microcontroller, developed strictly for educational purposes. It is designed to study communication protocols, analyze the RF spectrum, and debug smart home devices.

### ⚡ Features:
*   **UI:** Custom cyberpunk-style UI on ST7789 (1.14").
*   **WiFi & BLE:** Access Point, traffic analysis, connection management.
*   **RF Sub-GHz:** Sniffing and analyzing signals at 433/868 MHz (CC1101).
*   **IR Protocol:** Recording, playback, and brute-forcing IR signals.
*   **Web Control:** Built-in web server for monitoring via browser.
*   **Safety System:** Watchdog and Safe Mode to prevent lockups.
